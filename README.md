# AI Vendor Performance Analysis Tool

> Designed as a practical AI solution to evaluate vendor performance, identify gaps, and support governance decision-making.

## Overview
This project demonstrates an AI-driven workflow to analyze vendor performance data and generate structured insights for leadership and vendor management teams.

## Problem
Vendor performance data is often scattered across reports and lacks consistent analysis, making it difficult to assess SLA compliance, identify risks, and take corrective actions.

## Solution
This tool uses structured input and a controlled AI prompt to generate a standardized vendor performance report including:
- Performance Summary  
- SLA Compliance (actual vs contract)  
- Year-over-Year Comparison  
- Key Gaps  
- Risk Assessment  
- Prioritized Recommendations  

## Input
The system expects structured vendor performance data with fields such as:
- Vendor Name  
- Reporting Period  
- SLA Targets (contract)  
- Actual SLA Performance  
- MTTR  
- Response Time  
- Escalations (count + severity)  
- Previous Year Performance Summary  
- Additional Notes  

## Output
A structured and decision-ready report that:
- Highlights SLA deviations  
- Compares current vs historical performance  
- Identifies key performance gaps  
- Assesses business and operational risks  
- Provides prioritized recommendations  

## Value
- Improves vendor performance visibility  
- Enables data-driven governance decisions  
- Reduces manual analysis effort  
- Standardizes reporting across vendors  

## How it works
Structured Input → AI Prompt → Standardized Vendor Performance Report  

## Use Case
Designed for vendor managers, program leaders, and governance teams to evaluate vendor performance and drive improvement actions.
